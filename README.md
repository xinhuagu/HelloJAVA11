# Hello JAVA 11
[![Build Status](https://travis-ci.org/xinhuagu/HelloJAVA11.svg?branch=master)](https://travis-ci.org/xinhuagu/HelloJAVA11)

The project shows how to use maven to build module-based java 11 progam

1. buld the project:
```
./mvnw clean package
```
2. run the program:
```
./run.sh
````
