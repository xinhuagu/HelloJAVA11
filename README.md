# Hello JAVA 11
[![Build Status](https://travis-ci.org/xinhuagu/HelloJAVA11.svg?branch=master)](https://travis-ci.org/xinhuagu/HelloJAVA11)

The programe shows how to use maven to build module-based java 11 programe

1. buld the project:
```
mvn clean package
```
2. run the programe:
```
./run.sh
````
